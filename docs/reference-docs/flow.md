# Application Flow Document

## 2.1 High-Level User Flow
Launch App
   ↓
Overview Dashboard
   ↓
Data Exploration
   ↓
Scenario Simulation
   ↓
Before vs After Comparison
   ↓
Insights & Reporting

## 2.2 Page-Level Flow
### Overview

Display system purpose

Show global trade graph snapshot

Highlight high-level metrics

Data Explorer

Select country

View country attributes

Inspect trade partners

Explore historical trends

### Simulator (Core Flow)

Select country and commodity

Select disruption type

Set magnitude (bounded)

Run simulation

View before vs after results

Read uncertainty-aware explanation

### Insights

Vulnerability rankings

Critical trade routes

Cluster-level dependencies

Exportable reports

## 2.3 Error & Edge Case Handling

Invalid parameter selections are blocked

Unreliable data flagged with confidence warnings

Implausible simulations are labeled as low-confidence
